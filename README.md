# nuxt-onsenui-module
Onsen UI module for Nuxt.js

## Installation
```sh
$ npm i nuxt-onsenui-module
```

or

```sh
$ yarn add nuxt-onsenui-module
```

## Usage
Add `nuxt-onsenui-module` to `modules` of your `nuxt.config.js` file.
